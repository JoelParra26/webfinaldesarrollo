¿Qué es la Discalculia?

Definición

La Discalculia es una dificultad para aprender y entender conceptos matemáticos, como números, cantidades y operaciones.

Síntomas

  • Dificultad para entender conceptos matemáticos
  • Problemas para realizar operaciones básicas
  • Dificultad para recordar números y cantidades
